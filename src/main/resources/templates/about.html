About the Project

MobileEats is a Codeup capstone project created using several different technologies. Among these are: Java, Springboot, MySQL, Javascript, CSS, Filestack Api, Mapbox Api, Thymeleaf, and Jquery. The idea for this project was to create a web application that allows mobile food vendors to relay business information such as branding, current location, and menu items; while also giving user the ability to view, rate, and communicate with vendors in their area.
